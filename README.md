# Web Development Assignment 3

## Overview
This repository contains **Lab Assignment 3** for Web Development.  
It demonstrates the creation of a simple responsive web page using HTML and CSS.  
The project includes:
- `index.html` – the main webpage structure
- `style.css` – styling and responsive design
- `image.jpg` – example image used in the webpage

## Features
- Basic layout with header, content, and image
- CSS styling including colors, fonts, and spacing
- Responsive behavior using media queries

## Responsive Behavior
The webpage has been tested at the following breakpoints:
- **Desktop:** 1024px and above
- **Tablet:** 768px – 1023px
- **Mobile:** below 768px

Resize the browser window to observe changes in layout and styling.
